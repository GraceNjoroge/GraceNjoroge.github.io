    Virtual Pets Amok

    This project takes on a similar design to the Virtual-Pet-Shelter project with the addition of extended child classes for cat, organic dog and robot dog. Abstract classes and interfaces were added in this project to allow for multiple types of pets with varied needs and behaviors. The method calls were also expanded to not allow for certain types of pets to call on each other's methods i.e.Organic pet vs Robotic pet.

    This project utilized:

    • Java
      • OOP
      • TDD
    • Agile
    • Scrum
    Git Hub Repo

    Virtual Pet Shelter

    Used TDD to drive the creation of an application that simulates taking care of virtual pets in a shelter.The application includes a game loop that queries the user, then calls the appropriate method(s) on VirtualPetShelter and/or VirtualPet.

    This project utilized:

    • Java (TDD)
    • Agile
    • Scrum
    Github Repo

    Shopping Cart

    This project used an arraylist class to design a shopApp class that modeled shopping in a similar fashion to Amazon (where items can be added to a card and the total cost increases with each item added). A loop was used to simulate shopping.

    This project utilized:

    • Java (Array Lists, MAPS)
